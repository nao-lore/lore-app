1日の終了処理を一括実行してください。以下を順番に全て実行すること。

## Step 1: ファイル整理
1. ~/Downloads/ 直下の散らかりチェック:
   - スクショ → 中身をReadで確認してアセット/スクショ/ に分類・リネームして移動
   - 不要ファイル（インストーラ、重複、一時ファイル）→ 削除
   - lore-project関連ファイルが直下にあれば適切なフォルダに移動
2. ~/Downloads/lore-project/スクレイパー/ の中間ファイル（judge-*.json）→ 削除
3. 空フォルダがあれば削除

## Step 2: SESSION.md更新（セッション終了プロトコル）
session_protocol.md の「セッション終了時プロトコル」に従って全項目を実行:
1. 完了タスクを「今日の完了サマリー」に追記
2. 未完了タスクを該当セクションに追加
3. 新規作成ファイルがあればファイルマップに追加
4. 重要な決定があれば「最近の決定」に追加
5. 環境変更があれば「環境変更」に追加
6. `date '+%Y-%m-%d %H:%M'` で正確な日時を取得→最終更新日時を更新
7. セッション回数をインクリメント
8. 最新コミットハッシュを更新
9. メモ・発見の処理済みエントリを削除

## Step 3: 作業ログ追記
~/Downloads/lore-project/作業ログ.md に本日の作業内容を追記

## Step 4: Git自動コミット
```bash
cd ~/threadlog && git add -A && git commit -m "session end: $(date '+%Y-%m-%d %H:%M')"
```

## Step 5: ファイルマップ整合性チェック
SESSION.mdのファイルマップに記載されたファイルが実際に存在するか確認。
存在しないファイルがあれば修正。

## Step 6: メモリ更新
今日のセッションで学んだフィードバックや重要な決定があれば、メモリに保存。

## Step 7: 翌日の準備
「次にやること」セクションが空なら、job-search-plan.mdから次のDayタスクを記入。

## 完了報告
全ステップ完了後、「おやすみ！明日のタスクは〜」と簡潔に報告。
